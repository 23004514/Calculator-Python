# Scientific Calculator (Python)

## Description

This project is a **Scientific GUI Calculator built using Python and Tkinter**.
It provides basic arithmetic operations as well as scientific functions such as trigonometric calculations and logarithms. The calculator is designed with a simple graphical user interface (GUI) to make calculations easy and interactive.

## Features

* Basic arithmetic operations (+, −, ×, ÷)
* Trigonometric functions (sin, cos, tan)
* Logarithmic function (log10)
* Square root calculation
* Clear button to reset calculations
* User-friendly graphical interface

## Technologies Used

* Python
* Tkinter (for GUI)
* Math library (for scientific calculations)

## How to Run the Program

1. Install Python on your computer.
2. Download or copy the project files.
3. Open a terminal or command prompt.
4. Navigate to the project folder.
5. Run the program using:

python calculator.py

## Example Calculations

* sin(30) = 0.5
* log10(100) = 2
* √25 = 5

## Purpose of the Project

The purpose of this project is to demonstrate how **Python can be used to build graphical applications and perform scientific calculations**.
